# credit-risk
Bank Loan Applications – Credit Risk Management
Predicted the probability of loan default with 15% improved accuracy from a dataset of bank loan applications
o	Performed MySQL Nested Queries and Window Functions to explore the banking customers’ backgrounds.
o	Produced a clean dataset by extracting the raw data, imputing missing values, and encoding categorical variables.
o	Experimented with Logistic Regression and Random Forest models to enhance predictive performance by using the Area Under Receiver Operating Curve (AUROC).
